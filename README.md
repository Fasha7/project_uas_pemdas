# project_uas_pemdas
#  Aplikasi Rapor Sederhana  
**Project UAS Mata Kuliah Pemrograman Dasar**

Aplikasi Rapor Sederhana merupakan aplikasi desktop berbasis Python yang dirancang untuk membantu pengelolaan data siswa dan nilai mata pelajaran.  
Aplikasi ini mampu melakukan perhitungan nilai rata-rata, menentukan predikat nilai, serta menampilkan status kelulusan secara otomatis.

---

##  Gambaran Umum
Aplikasi ini dikembangkan untuk mempermudah proses pengolahan data akademik siswa tanpa memerlukan perhitungan secara manual.  
Seluruh data disimpan secara lokal sehingga dapat diakses kembali meskipun aplikasi telah ditutup.

---

##  Fitur Aplikasi
- Penambahan data siswa yang meliputi NISN, nama, dan kelas
- Validasi input data dan nilai siswa
- Pengisian dan pembaruan nilai mata pelajaran
- Perhitungan nilai secara otomatis, meliputi:
  - Nilai rata-rata
  - Predikat nilai (Aâ€“E)
  - Status kelulusan (LULUS atau TIDAK LULUS)
- Tampilan dashboard untuk ringkasan data siswa
- Fitur pencarian data siswa berdasarkan NISN atau nama
- Fitur pencarian nilai berdasarkan mata pelajaran
- Pengeditan dan penghapusan data siswa
- Pembuatan rapor siswa dalam format PDF

---

##  Cara Menggunakan Aplikasi
1. Buka folder project menggunakan **Visual Studio Code**
2. Jalankan file `main_rapor.py`
3. Pilih menu **Tambah Siswa** untuk memasukkan data siswa
4. Pilih menu **Input Nilai** untuk mengisi nilai mata pelajaran
5. Gunakan menu **Search Siswa** untuk:
   - Melihat detail data siswa
   - Mengedit atau menghapus data siswa
   - Mencetak rapor siswa ke dalam format PDF
6. Gunakan menu **Search Mapel** untuk melihat data nilai berdasarkan mata pelajaran
7. Lihat ringkasan keseluruhan data siswa pada menu **Dashboard**

---

##  Cara Menjalankan Program
1. Pastikan Python telah terinstal pada perangkat
2. Buka folder project menggunakan **Visual Studio Code**
3. Buka file `main_rapor.py`
4. Jalankan program menggunakan fitur **Run** yang tersedia di Visual Studio Code
