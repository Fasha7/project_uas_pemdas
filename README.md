**Project UAS Pemrograman Dasar**

Aplikasi Rapor Sederhana adalah aplikasi desktop berbasis Python yang digunakan untuk mencatat data siswa dan nilai pelajaran.  
Aplikasi ini membantu menghitung nilai rata-rata, menentukan predikat, serta menampilkan status kelulusan secara otomatis.

- Perhitungan otomatis:
